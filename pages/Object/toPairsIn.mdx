# `_.toPairsIn(object)`

Creates an array of own and inherited enumerable string keyed-value pairs for object which can be consumed by `_.fromPairs`. If object is a map or set, its entries are returned.

## Arguments

`object (Object)`: The object to query. 

## Returns

`(Array)`: Returns the key-value pairs.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
  var code = `var toPairsIn = require("lodash/toPairsIn");
function Foo() {
  this.a = 1;
  this.b = 2;
}
 
Foo.prototype.c = 3;
 
toPairsIn(new Foo);
// => [['a', 1], ['b', 2], ['c', 3]] (iteration order is not guaranteed)
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

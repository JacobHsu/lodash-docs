# `_.lowerFirst([string=''])`

Converts the first character of string to lower case.

## Arguments

`[string=''] (string)`: The string to convert.

## Returns

`(string)`: Returns the converted string.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
  var code = `var _ = require("lodash");
_.lowerFirst('Fred');
// => 'fred'
 
_.lowerFirst('FRED');
// => 'fRED'
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

<Demo />

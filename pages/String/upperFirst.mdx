# `_.upperFirst([string=''])`

Converts the first character of string to upper case.

## Arguments

`[string=''] (string)`: The string to convert.

## Returns

`(string)`: Returns the converted string.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
var code = `var _ = require("lodash");
_.upperFirst('fred');
// => 'Fred'
 
_.upperFirst('FRED');
// => 'FRED'
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

<Demo />

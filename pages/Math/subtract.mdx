# `_.subtract(minuend, subtrahend)`

Subtract two numbers.

## Arguments

`number (number)`: The first number in a subtraction.  
`subtrahend (number)`: The second number in a subtraction.

## Returns

`(number)`: Returns the difference.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
  var code = `var subtract = require("lodash/subtract");
subtract(6, 4);
// => 2
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

<Demo />

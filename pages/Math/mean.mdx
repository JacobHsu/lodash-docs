# `_.mean(array)`

Computes the mean of the values in array.

## Arguments

`array (Array)`: The array to iterate over.

## Returns

`(number)`: Returns the mean.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
  var code = `var mean = require("lodash/mean");
mean([4, 2, 8, 6]);
// => 5
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

# `_.sum(array)`

Computes the sum of the values in array.

## Arguments

`array (Array)`: The array to iterate over.

## Returns

`(number)`: Returns the sum.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
  var code = `var sum = require("lodash/sum");
sum([4, 2, 8, 6]);
// => 20
`;
  return (

    </div>
  );
};

# `_.ceil(number, [precision=0])`

Computes number rounded up to precision.

## Arguments

`number (number)`: The number to round up.   
`[precision=0] (number)`: The precision to round up to.  

## Returns

`(number)`: Returns the rounded up number.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
  var code = `var ceil = require("lodash/ceil");
ceil(4.006);
// => 5
 
ceil(6.004, 2);
// => 6.01
 
ceil(6040, -2);
// => 6100
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

<Demo />

# `_.add(augend, addend)`

Adds two numbers.

## Arguments

`augend (number)`: The first number in an addition.  
`addend (number)`: The second number in an addition.

## Returns

`(number)`: Returns the total.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
  var code = `var add = require("lodash/add");
add(6, 4);
// => 10
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

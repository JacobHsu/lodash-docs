# `_.bind()`

`_.bind(func, thisArg, [partials])`

Creates a function that invokes func with the this binding of thisArg and partials prepended to the arguments it receives.

The `_.bind.placeholder` value, which defaults to _ in monolithic builds, may be used as a placeholder for partially applied arguments.

**Note**: Unlike native Function#bind, this method doesn't set the "length" property of bound functions.

## Arguments
`func (Function)`: The function to bind.  
`thisArg (*)`: The this binding of func.  
`[partials] (...*)`: The arguments to be partially applied.

## Returns
`(Function)`: Returns the new restricted function.

## Example

import Embed from 'runkit-embed-react';

export const Demo = () => {
var code =  `var bind = require("lodash/bind");
function greet(greeting, punctuation) {
  return greeting + ' ' + this.user + punctuation;
}
 
var object = { 'user': 'fred' };
 
var bound = bind(greet, object, 'hi');
bound('!');
// => 'hi fred!'
`   
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

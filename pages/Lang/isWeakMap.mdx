# `_.isWeakMap(value)`

Checks if `value` is classified as a WeakMap object.

## Arguments

`value (*)`: The value to check. 

## Returns

`(boolean)`: Returns `true` if `value` is weak map, else `false`.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
var code = `var isWeakMap = require("lodash/isWeakMap");
 
isWeakMap(new WeakMap); 
// => true
 
isWeakMap(new Map);
// => false
 
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

<Demo />

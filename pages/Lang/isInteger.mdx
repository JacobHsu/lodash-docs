# `_.isInteger(value)`

Checks if `value` is an integer.  
**Note**: This method is based on Number.isInteger.

## Arguments

`value (*)`: The value to check.  

## Returns

`(boolean)`: Returns `true` if `value` is an integer, else `false`.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
var code = `var isInteger = require("lodash/isInteger");
 
isInteger(3);
// => true

isInteger('3');
// => false
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

# `_.isMap(value)`

Checks if `value` is classified as a `Map` object.

## Arguments

`value (*)`: The value to check.  

## Returns

`(boolean)`: Returns `true` if `value` is a valid length, else `false`.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
var code = `var isMap = require("lodash/isMap");
 
isMap(new Map);
// => true
 
isMap(new WeakMap);
// => false
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

<Demo />

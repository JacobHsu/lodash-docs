# `_.lte(value)`

Checks if value is less than or equal to other.

## Arguments

`value (*)`: The value to compare.  
`other (*)`: The other value to compare.

## Returns

`(boolean)`: Returns `true` if `value` is less than or equal to other, else `false`.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
  var code = `var lte = require("lodash/lte");
 
lte(1, 3);
// => true
 
lte(3, 3);
// => true
 
lte(3, 1);
// => false
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

<Demo />

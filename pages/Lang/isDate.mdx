# `_.isDate(value)`

Checks if value is classified as a `Date` object.

## Arguments

`value (*)`: The value to check.

## Returns

`(boolean)`: Returns `true` if `value` is a date object, else `false`.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
  var code = `var isDate = require("lodash/isDate");
isDate('Mon April 23 2012');
// => false
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

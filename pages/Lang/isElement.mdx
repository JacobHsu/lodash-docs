# `_.isElement(value)`

Checks if value is likely a DOM element.

## Arguments

`value (*)`: The value to check.

## Returns

`(boolean)`: Returns `true` if `value` is a DOM element, else `false`.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
  var code = `var isElement = require("lodash/isElement");
isElement(document.body);
// => true
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

# `_.isWeakSet(value)`

Checks if `value` is classified  as a WeakSet object.

## Arguments

`value (*)`: The value to check. 

## Returns

`(boolean)`: Returns `true` if `value` is a weak set, else `false`.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
var code = `var isWeakSet = require("lodash/isWeakSet");
 
isWeakSet(new WeakSet);
// => true
 
isWeakSet(new Set);
// => false
 
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

<Demo />

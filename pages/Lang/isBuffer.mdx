# `_.isBuffer(value)`

Checks if value is a buffer.

## Arguments

`value (*)`: The value to check.

## Returns

`(boolean)`: Returns `true` if `value`  is a buffer, else `false`.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
  var code = `var isBuffer = require("lodash/isBuffer");
isBuffer(new Buffer(2));
// => true
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

<Demo />

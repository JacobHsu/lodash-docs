# `_.isNaN(value)`

Checks if value is NaN.

**Note**: This method is based on `Number.isNaN` and is not the same as global isNaN which returns true for undefined and other non-number values.

## Arguments

`value (*)`: The value to check. 

## Returns

`(boolean)`: Returns `true` if `value` is  `NaN`, else `false`.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
var code = `var isNaN = require("lodash/isNaN");
 
isNaN(NaN);
// => true
 
isNaN(new Number(NaN));
// => true
 
isNaN(undefined);
// => true
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

# `_.isFinite(value)`

Checks if value is a finite primitive number.

**Note**: This method is based on `Number.isFinite`.


## Arguments

`value (*)`: The value to check.  

## Returns

`(boolean)`: Returns `true` if `value` is a finite number, else `false`.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
var code = `var isFinite = require("lodash/isFinite");

isFinite(3);
// => true

isFinite('3');
// => false
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

# `_.isRegExp(value)`

Checks if `value` is classified as a `RegExp` object.

## Arguments

`value (*)`: The value to check. 

## Returns

`(boolean)`: Returns `true` if `value` is a regexp, else `false`.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
var code = `var isRegExp = require("lodash/isRegExp");
 
isRegExp(/abc/);
// => true
 
isRegExp('/abc/');
// => false
 
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

<Demo />

# `_.isSet(value)`

Checks if `value` is classified as a Set object.

## Arguments

`value (*)`: The value to check. 

## Returns

`(boolean)`: Returns `true` if `value` is a set, else `false`.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
var code = `var isSet = require("lodash/isSet");
 
isSet(new Set);
// => true
 
isSet(new WeakSet);
// => false
 
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

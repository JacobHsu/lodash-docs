# `_.isError(value)`

Checks if value is an Error, EvalError, RangeError, ReferenceError, SyntaxError, TypeError, or URIError object.

## Arguments

`value (*)`: The value to check.  

## Returns

`(boolean)`: Returns `true` if value is an `error` object, else `false`.  

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
var code = `var isError = require("lodash/isError");

isError(new Error);
// => true
 
isError(Error);
// => false

`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

<Demo />

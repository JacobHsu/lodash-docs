# `_.isNull(value)`

Checks if `value` is `null`.

## Arguments

`value (*)`: The value to check. 

## Returns

`(boolean)`: Returns `true` if `value` is `null`, else `false`.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
var code = `var isNull = require("lodash/isNull");
 
isNull(null);
// => true
 
isNull(void 0);
// => false
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

# `_.isFunction(value)`

Checks if `value` is classified as a `Function` object.

## Arguments

`value (*)`: The value to check.  

## Returns

`(boolean)`: Returns `true` if `value` is a function, else `false`.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
var code = `var isFunction = require("lodash/isFunction");
 
isFunction(/abc/);
// => false
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

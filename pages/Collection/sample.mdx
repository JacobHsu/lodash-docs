# `_.sample(collection)`

Gets a random element from collection.

## Arguments
`collection (Array|Object)`: The collection to sample.

## Returns
`(*)`: Returns the random element.

## Example

import Embed from 'runkit-embed-react';

export const Demo = () => {
var code =  `var sample = require("lodash/sample");
sample([1, 2, 3, 4]);
`   
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

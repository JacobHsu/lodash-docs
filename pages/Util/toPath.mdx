# `_.toPath(value)`

Converts value to a property path array.

## Arguments

`value (*)`: The value to convert.

## Returns

`(Array)`: Returns the new property path array.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
var code = `var _ = require("lodash");
_.toPath('a.b.c');
// => ['a', 'b', 'c']
 
_.toPath('a[0].b.c');
// => ['a', '0', 'b', 'c']
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};

# `_.prototype.value()`

Executes the chain sequence to resolve the unwrapped value.

## Aliases

`_.prototype.toJSON`, `_.prototype.valueOf`

## Returns

`(*)`: Returns the resolved unwrapped value.

## Example

import Embed from "runkit-embed-react";

export const Demo = () => {
  var code = `var _ = require("lodash");
_([1, 2, 3]).value();
// => [1, 2, 3]
`;
  return (
    <div>
      <Embed source={code} />
    </div>
  );
};
